# Rust CHIP 8 Emulator

Chip 8 emulator written in Rust. Graphics and audio use SDL2, statically linked.

Usage: `rust_chip8 rom_file`

## Keypad mappings

```
Host                   Emulator
1 2 3 C                1 2 3 4
4 5 6 D                Q W E R
7 8 9 E                A S D F
A 0 B F                Z X C V
```

## ROMS
Many roms can be found at https://github.com/dmatlack/chip8/tree/master/roms.
